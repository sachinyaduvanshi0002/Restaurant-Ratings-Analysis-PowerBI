# Restaurant Ratings Analysis Dashboard (Power BI)

## Overview

This project presents a data analysis dashboard that explores restaurant ratings and customer engagement using Power BI. The analysis focuses on understanding rating distribution, identifying rating trends, and evaluating customer interaction through vote counts.

This dashboard was developed during my **Data Analysis Internship at Cognifyz Technologies** as part of **Level 2 – Task 1** of the internship program.

## Objective

The objective of this project is to analyze restaurant data and gain insights into:

* Distribution of restaurant aggregate ratings
* Most common rating ranges
* Average number of votes received by restaurants
* Customer engagement patterns based on ratings

## Tools & Technologies

* Power BI
* Data Visualization
* Data Analysis
* CSV Dataset

## Dashboard Features

The Power BI dashboard includes the following visualizations:

* **Aggregate Rating Distribution**

  * Displays the distribution of restaurant ratings across different rating ranges.

* **Average Votes Indicator**

  * Shows the average number of votes received by restaurants, representing customer engagement.

* **Restaurant Rating Share**

  * A visual representation of rating proportions across restaurants.

* **Locality Filter**

  * Allows users to filter restaurant data based on locality to explore regional performance.

## Key Insights

* A large number of restaurants fall within the mid-to-high rating range.
* Restaurants with higher ratings often receive more votes, indicating stronger customer engagement.
* Locality-based filtering helps identify areas with better-performing restaurants.
